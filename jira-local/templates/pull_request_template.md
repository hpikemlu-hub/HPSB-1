# Ringkasan
Jelaskan perubahan yang dilakukan, latar belakang, dan hasil yang diharapkan.

# Issue Terkait
Tautkan issue dengan format: HPI-xxx
Contoh: Fixes HPI-16, Relates to HPI-2

# Checklist
- [ ] Lulus lint & typecheck
- [ ] Unit/komponen test berjalan (jika ada)
- [ ] Smoke test area terkait lulus
- [ ] Perubahan RLS tidak mengekspos data
- [ ] Error states & loading states tertangani
- [ ] Dokumentasi perubahan & langkah verifikasi

# Bukti & Catatan
- Screenshot/log/rekaman singkat bila perlu
- Catatan rollback bila diperlukan
